### 标签只读属性

`readonly="readonly"` 值不能修改，仍然可以使用tab键切换到该字段，还可以选中或拷贝其文本
    
`disabled="disabled"` 禁用input元素

### video标签--[参考文档](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/video)

```
<video src="{video-url}" 
    preload="auto" 
    controls="controls" 
    autoplay="autoplay" 
    playsinline="true" 
    webkit-playsinline="true" 
    x5-video-player-type='h5' 
    x5-video-player-fullscreen='true' 
    poster="imgSrc" 
    height="180px">
    </video>
```
